# 🧠 AI Resume Builder

An intelligent resume builder powered by MERN Stack and Google Gemini AI that helps job seekers craft professional, ATS-friendly resumes with AI assistance.


## 🚀 Features

- 🔐 User Authentication — Secure login/signup; only authenticated users can create and manage resumes
- ✍️ AI-Enhanced Professional Summary — Gemini AI rewrites your summary to sound polished and professional
- 💼 AI-Improved Experience Section — Transform bullet points into impactful, achievement-driven descriptions
- 🎯 ATS-Friendly Skills Optimization — AI suggests and replaces skills with ATS-optimized keywords to beat applicant tracking systems
- 📄 Resume Creation & Management — Create, edit, and manage multiple resumes from your dashboard

# 🛠️ Tech Stack

| Layer | Technology |
|--------|-------------|
| Frontend | Reactjs,Tailwindcss |
| Backend | Nodejs Expressjs |
| Database | MongoDB |
| AI API | Google gemini ai |
| Auth | JWT |
| state management | Zustand |
| Deployement | Render,Vercel |
